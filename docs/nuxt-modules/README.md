# Nuxt Modules registry

To list **Nuxt Desktop** on [nuxt.com/modules/desktop](https://nuxt.com/modules/desktop):

1. Review [`desktop.yml`](./desktop.yml).
2. Fork [nuxt/modules](https://github.com/nuxt/modules).
3. Add `modules/desktop.yml` and an icon under `public/icons/` (add `desktop.svg` — OWD logo or simple monitor icon).
4. Open a pull request, or file a [module request issue](https://github.com/nuxt/modules/issues/new?template=module_request.yml).

See [NUXT_DESKTOP_ECOSYSTEM.md](../NUXT_DESKTOP_ECOSYSTEM.md) for the full roadmap including `@nuxtjs/desktop`.
